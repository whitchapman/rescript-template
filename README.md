# ReScript Template

This is a manual recreation of the official ReScript [starter template](https://github.com/rescript-lang/rescript-project-template) as of 625edc7.

## Installation

```sh
npm install
```

## Build

- Build: `npm run build`
- Build & watch: `npm run build:watch`
- Clean: `npm run clean`

## Run

```sh
node src/Demo.mjs
```
